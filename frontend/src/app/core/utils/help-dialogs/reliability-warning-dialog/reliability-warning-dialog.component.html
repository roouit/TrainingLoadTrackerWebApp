Huomautus datan epätarkkuudesta
Tämä huomautus näytetään, koska kroonisen kuormituksen laskentaan käytettävä aikaväli on pidempi, kuin aika ensimmäisestä syöttämästäsi harjoituksesta.
Tämä otetaan huomioon laskennassa, mutta kuormittumisen arvojen luotettavuus on madaltunut rajoitetun datan määrän takia.
Mitä vähemmän dataa on, sitä herkemmin kuormittumisen arvot muuttuvat yhdestä harjoituksesta.
